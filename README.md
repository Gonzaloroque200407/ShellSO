## Instalação

Para instalar o projeto, siga os passos abaixo:


Clone o repositório
```
git clone https://github.com/seuusuario/seu-repositorio.git
```
Navegue até o diretório do projeto
```
cd seu-repositorio
```

Instale as dependências

```
npm install
```
